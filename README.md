# 📝 App ToDo Flutter

Ứng dụng quản lý công việc đơn giản xây dựng bằng Flutter + SQLite.

---

## 📲 Dành cho người dùng (Download APK)

1. Vào tab [Releases](https://github.com/chikien-finn/App_ToDo_flutter/releases)
2. Download file `app-release.apk`
3. Cài vào điện thoại Android (cho phép cài từ nguồn không rõ nếu được hỏi)

> ⚠️ Yêu cầu Android 5.0 trở lên

---

## 🛠️ Dành cho Developer (Clone & Build)

### Yêu cầu
- Flutter SDK >= 3.0.0
- Dart >= 3.0.0
- Android Studio hoặc VS Code

### Cài đặt

```bash
git clone https://github.com/chikien-finn/App_ToDo_flutter.git
cd App_ToDo_flutter/todo_app
flutter pub get
flutter run
```

### Build APK

```bash
flutter build apk --release
```

File output: `build/app/outputs/flutter-apk/app-release.apk`
